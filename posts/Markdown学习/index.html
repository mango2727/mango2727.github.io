Markdown

1.1什么是Markdown

Markdown时一种轻量级的标记语言，允许使用纯文本格式编写文档后转为有效的HTML文档，Markdown编写的文档可以导出HTML、Word、图像和PDF等多种格式的文档，编写的文档后缀为.md或.markdown。

它的优点在于：
纯文本编辑、简单易上手、支持跨平台同步数据、可插入图片视频、随时修改不用担心word等工具出现排版错误

Markdown基本语法

2.1 标题

2.1.1写法一

使用#标记，可以表示1-6级标题，#个数越多字体越大，一级标题最大，六级标题最小

注意：在编写时记得#与字符之间留一个空格，并且标题应置于首，放入表格中可能无法正常解析

代码如下：
#一级标题
##二级标题
###三级标题
####四级标题
#####五级标题
######六级标题
